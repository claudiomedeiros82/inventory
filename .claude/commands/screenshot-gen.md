Plan the App Store and Play Store screenshots for this app.

Read the app to understand what screens exist:
- `lib/features/` — what screens are available
- `pubspec.yaml` — app name
- `docs/aso-strategy.md` if it exists (generated by `/aso-optimize`)

Delegate to the Screenshot Generator agent.

Produce a complete screenshot plan:

**For each of 5 screenshots:**
- Which screen of the app to capture
- Caption text (max 5 words, benefit-led)
- Background color or gradient suggestion
- What UI element to highlight or zoom into
- Any annotation or callout to add

**Play Store feature graphic (1024×500):**
- Tagline
- Visual concept

**Device frame guidance:**
- Frame or no frame recommendation for this app type
- Which device mockup to use

**Localization:**
- If targeting non-English markets: caption alternative for each screenshot in relevant language
- Any cultural considerations

**Tools to create them:**
- DaVinci Resolve / Figma / Canva — which fits the dev's workflow
- `flutter screenshot` command for capturing the actual screens
- Frame mockup sources

After the plan, ask:
"Do you want me to write the caption copy for all 5 screenshots? If you're targeting non-English markets, specify which languages."

If yes, produce finalized copy ready to paste into the store listing.
